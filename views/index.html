Usage examples

Enhance Mode

Get AI-powered enhancements and expansions for your prompts

curl -X POST "http://localhost:3000/enhance?projectId=YOUR_PROJECT_ID" \
  -H "Content-Type: application/json" \
  -H "Authorization: Bearer $FIREBASE_ID_TOKEN" \
  -d '{"message": "Let's add Kafka for no apparent reason"}' \
  -c cookies.txt -b cookies.txt

Ask Mode

Get critical analysis with probing questions

curl -X POST "http://localhost:3000/ask?projectId=YOUR_PROJECT_ID" \
  -H "Content-Type: application/json" \
  -H "Authorization: Bearer $FIREBASE_ID_TOKEN" \
  -d '{"message": "What are the likely implications of adding Kafka when we obviously don't need it?"}' \
  -c cookies.txt -b cookies.txt
